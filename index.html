University of Wolverhampton - BSc Cyber Security

About This Project

This project demonstrates common web security vulnerabilities (SQL Injection and XSS) and their mitigation techniques through two interactive versions.

⚠️ Insecure Version

  • No input validation
  • Vulnerable to XSS attacks
  • Simulated SQL injection
  • Dangerous output methods
  • Insecure client-side storage & logging
  • Reflected input from URL parameters

✓ Secure Version

  • Input validation
  • XSS protection
  • SQL injection prevention
  • Safe output encoding
  • Password strength indicator
  • CSRF token & basic rate-limit simulation
